Intro

Thank you for purchasing this theme. This file will guide you through on how to style and briefly modify your theme. This documentation is rather brief and covers the basics of the theme. If you need support please contact me through my profile page on ThemeForest.

Install

Onve you've downloaded your theme to install it you will need to upload the folder "swiss" (located inside the "vanilla" folder) in your themes folder.

Open your FTP and upload the wntire folder "swiss" (just this folder) inside your themes folder located in your vanilla root directory.

ie: Folder /vanilla/swiss is uploaded into public_html/[vanilla root folder]/themes

once the folder has been completely uploaded log in into your vanilla install as an administrator and go to your Dashboard. Click on "Themes" on your left sidebar and Apply the Swiss Theme.

Your theme is now activated.

Theme Options

Swiss comes with two theme options. You'll notice once the theme is installed and activated a Theme Options menu appears right below the Themes menu. Bly clicking there you'll be able to fill two options.

1) Colour - Vanilla has 4 different colour option to choose from. If you wishto modify these colours please refer to the Files section.

2) Logo URL - Insert here a URL for your logo image. Upload your logo image on your server or in a free hosting service such as imageshack.us and paste your image URL here (maximum 180px wide).

Click save and you're good to go!

The rest of the theme is based on vanilla options and should be referred to the vanilla documentation.

Files

The download file comes with various different files.

PSD Folder - Contains the PSD files in case you wish to make modifications on graphics.

vanilla Folder - Contains your theme files.

Help - Contains your documentation (this file)

Theme FIles

If you wish to modify your theme the .css files are inside the "design" folder. each custom_%.css file correponds to a colour whereas style.css is the main style file.

If you wish to mosify something in the body of the theme like the strcture, go to your views folder and edit the file wished. To know what files does what please refer to the vanilla documentation.

Customizing your theme

If you want to customize your theme with your own colours, pick a color (like green for instance) and open your file vanilla/swiss/design/custom_green.css and change the colours in there. This will only change the green colours. If you want to change the rest of the colour you should edit hte style.css file instead.
